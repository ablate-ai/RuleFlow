## 快速导航

### 核心功能

- **协议解析**：支持 Trojan、VMess、VLESS、Shadowsocks、Hysteria2、TUIC、AnyTLS 等主流协议
- **配置生成**：支持 Clash Meta、Stash、Surge、Sing-Box 格式
- **规则模板**：支持自定义 YAML/INI 模板和 `__NODES__` 占位符
- **节点过滤**：支持按正则表达式过滤节点到代理组
- **链式代理**：支持 `dialer-proxy` 链式代理（中转落地）

### 技术栈

- **后端**：Go 1.24+
- **数据库**：PostgreSQL
- **缓存**：Redis（可选）
- **前端**：HTML + CSS + JavaScript（原生）
- **部署**：Docker + Docker Compose

### 常用入口

```
main.go                  # 程序入口
internal/app/parser.go   # 协议解析器
api/handlers.go          # API 处理器
services/                # 业务服务
database/                # 数据访问层
web/                     # 前端页面
```

## 项目概览

RuleFlow 是一个**代理订阅转换服务**，主要功能：
1. 拉取各种协议的代理订阅
2. 解析为统一格式的节点
3. 根据模板生成配置文件
4. 提供 Web 管理界面和 REST API
5. 支持多客户端配置输出

通过灵活的模板系统和强大的节点过滤功能，帮助用户快速生成符合需求的代理配置。
